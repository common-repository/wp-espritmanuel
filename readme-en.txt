== Description ==

wp-espritmanuel allowing bloggers to easily embed step-by-step guides from EspritManuel.com. Just go to any Espritmanuel guide page and copy the guide-Id. Click on the new editor button "espritmanuel" and Paste that id. It will automatically be converted to an embedded EspritManuel player. 

== Installation ==

This section describes how to install the plugin and get it working.

= Method 1: Quick-install via WordPress Admin =

1. From your Admin Dashboard, click on "Plugins" in the sidebar.
2. Then click "Add New" at the top of the page.
3. Type "EspritManuel" in the search box and click "Search Plugins"
4. Click "Install Now" under *EspritManuel Embed* - The plugin will download and install to your server.
5. On the next screen, click "Activate Plugin"
6. Go to [EspritManuel](http://www.espritmanuel.com "EspritManuel") and to your guide page and copy the GuideId (e.g. 50fab0948523eebe100001b4)
7. Paste the id in a blog post or a page. Make sure the URL is on its own line and not hyperlinked.

= Method 2: Advanced install via FTP =

1. Upload the `espritmanuel_embed_wp` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to [EspritManuel](http://www.espritmanuel.com "EspritManuel") and to your guide page and copy the GuideId (e.g. 50fab0948523eebe100001b4)
4. Paste the GuideId in a blog post or a page. Make sure the URL is on its own line and not hyperlinked.

= Can I embed a EspritManuel player on a blog hosted on WordPress.COM? =

Currently the "hosted" version of WordPress does not support many external espritmanuel guides and won't let you install themes or plugins. If you're using the WordPress.com service, why not move your blog to a different host? 
Or you can ask WordPress to support the EspritManuel player Embed. Contact WordPress here: http://en.support.wordpress.com/contact/
