== Description ==

wp-espritmanuel permet aux blogueurs d'insérer facilement un guide étape-par-étape provenant du site espritmanuel.com dans leurs articles. Il suffit d'aller sur la page d'un guide sur espritmanuel.com et
de récupérer l'identifiant du guide en cliquant sur le bouton "insérer sur votre blog". Une fois sur votre editeur wordpress, un nouveau bouton "EspritManuel" vous permettra d'insérer votre guide. 
Le contenu du guide sera directement intégré à votre article.

== Installation ==

Pour installer ce plugin sur votre wordpress, il y a 2 méthodes.

= Méthode 1: Installation rapide via l'interface d'administration =

1. Depuis votre interface d'administration wordpress, cliquer sur "Extensions" dans le menu de gauche.
2. Ensuite, cliquer sur Ajouter en haut de la page.
3. Saisir "EspritManuel" dans la zone de recherche et cliquer sur le bouton "Chercher parmis les extensions"
4. Cliquer sur "Installer maintenant" juste en dessous de l'extension *wp-EspritManuel* - L'extension sera téléchargée et installée sur votre serveur.
5. Sur l'écran suivant, cliquer sur "activer l'extension"

= Method 2: Méthode Avancée par FTP =

1. Uploader le dossier wp-espritmanuel dans le dossier `/wp-content/plugins/` du serveur
2. Activer l'extension dans la liste des extension de votre wordpress.

== Utilisation ==

1. Aller sur [EspritManuel](http://www.espritmanuel.com "EspritManuel") et récupérer l'identifiant du guide à insérer (ex. 50fab0948523eebe100001b4)
2. Lors de la rédaction d'un article, cliquer sur le bouton "EspritManuel" de votre editeur Wordpress.
3. Coller l'identifiant dans la popup. Choississez la largeur que prendra la guide sur votre page, c'est terminé !


== Frequently Asked Questions ==

= Qu'est-ce que EspritManuel ? =

[EspritManuel est un service internet permettant de créer très facilement des guides photos étapes-par-étapes pour vos projet de bricolage ou vos projets DIY](http://www.espritmanuel.com/ "EspritManuel")

= Est-ce que je peux intégrer un guide EspritManuel sur un blog hébergé par Wordpress.COM =

Actuellement, la version hébergée sur Wordpress.com ne supporte pas l'ajout de thèmes ou d'extensions. Si vous utilisez  WordPress.com, nous vous conseillons d'héberger vous même votre blog wordpress chez un autre hébergeur. C'est simple et pas cher.
Vous pouvez également essayer de demander à Wordpress.com de supporter l'extension espritmanuel en les contactant: http://en.support.wordpress.com/contact/
